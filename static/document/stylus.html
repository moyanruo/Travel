Stylus

Stylus 与 less,sass 一样 也是一种 css 编译语言

Stylus 出现的比较晚，因此语法比较新

Stylus 可以解决一些 less,sass 中的语法问题

Stylue 中，认为 对于元素样式的设置 ：; {} 都是无意义的，所以在Stylus中不用书写他们

Stylus 中的空格、tab 换行 空白符 都是很重要的

浏览器不认识 Stylus，所以必须将Stylus 编译成，Stylus 也是 用js 写的，因此很多 js 方法 都可以在 Stylus 中使用

安装：npm install stylus --save

npm install stylus-loader --save
